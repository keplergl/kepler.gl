# Gleich-Brother deckgl app  

This is the src code of the Gleich-Brother web-app. 

#### 1. Install

```sh
npm install
```

or

```sh
yarn
```


#### 2. Mapbox Token
add mapbox access token to node env

```sh
export MapboxAccessToken=<your_mapbox_token>
```

#### 3. Start the app

```sh
npm start
```
