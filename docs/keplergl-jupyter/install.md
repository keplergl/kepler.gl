# Installation

## Prerequisites

- Python >= 3.9
- JupyterLab >= 4.0 or Notebook >= 7.0

## Install with pip

```bash
$ pip install keplergl
```

The widget is built on [anywidget](https://anywidget.dev/) and works out of the box in modern Jupyter environments — no extra `nbextension` or `labextension` install steps are needed.
